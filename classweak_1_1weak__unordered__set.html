Detailed Description

template<class Key, class Hash = std::hash<Key>, class KeyEqual = std::equal_to<>, class Allocator = std::allocator<Key>>
class weak::weak_unordered_set< Key, Hash, KeyEqual, Allocator >

An unordered set of weak pointers.

Uses key equality, not pointer equality, even though the keys are held behind pointers.
